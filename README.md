# Калькулятор выбросов и сокращений выбросов ПГ — ТЭЦ-ПВС, КА-11 (ПАО «Северсталь»)

Веб-приложение для расчёта выбросов парниковых газов по проектному и базовому сценариям,
сокращения выбросов и отклонений от проектной документации климатического проекта
«Полезная утилизация вторичных энергетических ресурсов доменного газа с выработкой
электроэнергии».

Расчёт выполняется по формулам **(1)–(13)** Плана мониторинга проектной документации (PDD)
и Приказа Минприроды России от 27.05.2022 № 371.

## Онлайн-версия

Приложение развёрнуто на Streamlit Community Cloud — ссылка добавляется после деплоя.

## Локальный запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Состав

- `app.py` — веб-интерфейс (Streamlit);
- `engine.py` — расчётное ядро (формулы 1–13, параметры П1…П31, данные ПДД);
- `requirements.txt` — зависимости.

## Проверка корректности

При исходных данных проектной документации приложение воспроизводит плановые итоги PDD за
2025–2034 гг.: ΣPE ≈ 1 018 377; ΣBE ≈ 3 509 570; **ΣER ≈ 2 491 193 т CO₂-экв.**
(таблица 8.4.1 проектной документации).
